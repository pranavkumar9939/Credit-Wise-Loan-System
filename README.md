# CreditWise: Loan Approval Prediction System

CreditWise is a Machine Learning project designed to predict loan approval decisions using historical financial data. The model analyzes multiple applicant attributes such as income, age, savings, collateral value, loan amount, loan term, debt-to-income ratio, and credit score to determine whether a loan should be approved or rejected.

The goal of this project is to demonstrate how machine learning can assist financial institutions in making faster, data-driven, and unbiased lending decisions.
